whxiaoyi-cloud 基于spring-cloud-alibaba构建的微服务生态，减少重复造轮子

#### 使用说明

1.  git clone https://gitee.com/whxiaoyu123/whxiaoyu-cloud
2.  mvn clean package
3.  java -jar whxiaoyu-**.jar
